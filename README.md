# セルオートマトン

## packages
- `ca`: 状態が$[0,1]$で隣接セルと相互作用する基本的モデル。
    - `CA.java`: 1次元セルオートマトン
    - `CAUtils.java`: 関連するユーティリティなど
- `rule184`: 最も簡単な交通流モデルであるルール184について、流量や速度を計算する
    - `Observable.java`: 平均値を求めるための抽象クラス
    - `Flow.java`: 平均流量を求め、基本図を出力
    - `Speed.java`: 平均速度を求め、密度との関係を出力
- `gui`: セルオートマトンのシミュレーションを行うGUI
